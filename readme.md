4.游客转动方式

* 同一景区内

  - 景点间寻路
  - 从景点找出口

* 不同景区间

  - 假设A景区甲景点到B景区乙景点，应为从甲景点找A景区出口，A景区出口到B景区入口，B景区入口到乙景点

  - 因此，跨景区景点在数据中不联通，以保证寻路是必经出入口
